## üîÑ Corrected Workflows

### **Customer Workflow (CORRECT)**

```
1. Register ‚Üí Creates User + CustomerProfile
2. Browse Fabrics ‚Üí GET /fabrics
3. Select Fabric ‚Üí GET /fabrics/:id/customize
4. Customize Design:
   - Upload custom image (Multer)
   - Add text, select colors, size
   - System creates Customization
   - Auto-adds to Cart
5. View Cart ‚Üí GET /customer/cart
6. Checkout ‚Üí POST /customer/checkout (creates Order: status=pending, paymentStatus=unpaid)
7. Pay ‚Üí POST /customer/pay/:id (sets paymentStatus=paid) [DUMMY - no real gateway]
8. Track Order ‚Üí GET /customer/orders/:id
```

### **Designer Workflow (CORRECT)**

```
1. Register ‚Üí Creates User + DesignerProfile (status="pending")
2. Wait for Admin Approval ‚Üí GET /designer/pending
3. Admin Approves ‚Üí DesignerProfile.status = "approved"
4. Create Templates ‚Üí POST /designer/templates (upload template image)
5. View Earnings ‚Üí GET /designer/dashboard
   - Earnings = DesignTemplate.usageCount √ó commissionRate ($5)
```

### **Manager Workflow (CORRECT)**

```
1. View Dashboard ‚Üí GET /manager/dashboard
   - Orders grouped by status (pending, in_production, completed, etc.)
2. Start Production ‚Üí POST /manager/orders/:id/start
   - Validate: status=pending && paymentStatus=paid
   - Set status=in_production, productionStartedAt=now
3. Mark Complete ‚Üí POST /manager/orders/:id/complete
   - Validate: status=in_production
   - Set status=completed, productionCompletedAt=now
4. Ship Order ‚Üí POST /manager/orders/:id/ship
   - Validate: status=completed
   - Set status=shipped, shippedAt=now
5. Mark Delivered ‚Üí POST /manager/orders/:id/deliver
   - Set status=delivered, deliveredAt=now
```

### **Admin Workflow (CORRECT)**

```
1. Dashboard ‚Üí GET /admin/dashboard (stats, pending designers)
2. Approve Designers ‚Üí POST /admin/designers/:id/approve
3. Manage Fabrics ‚Üí CRUD operations on Fabric model
4. Manage Users ‚Üí Block/unblock users
```
